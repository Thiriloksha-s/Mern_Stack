CSS - Cascaded Style Sheet

It is used to style your respective web page

Types

  • Inline (easiest styling) - within the tag
  • Internal - written within the page
  • External - It is written separately or outside the page

link tag is an important tag in external css and also it is also useless when it is not used

Selectors

  • Class - ( . )
  • id - ( # )
  • tag(element) - tag
  • Universal - ( * )

Inline css overrides the internal css as inline holds high priority than internal css

Styling priorities

    Type 1

  • internal - priority 2
  • inline - priority 1

  • external - priority 2
  • inline - priority 1

  • external

Intermediate CSS

content does not move

padding moves outwards from content

margin moves towards the content

border moves towards the content
