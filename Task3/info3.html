MERN Stack - Intro

MERN is a technology containing tools such as MongoDb, Express, React, Node js

  1. Node js - runtime environment for javascript (nvm- Node Visual Machine)
  2. React - It is a frontend development package
  3. 3. Express - It is a backend development package
  4. MongoDB - It is a database which is used in backend storage

Backend ——> Combination of Express and database (Express+database)

VCS - Version Control System

Types

  • Distributed VCS
  • Centralized VCS

Example : 3.41.0

  • 1 st number : 3 --> Major version
  • 2 nd number : 41 --> Minor version
  • 3 rd number : 0 --> Patch/bug version

Git and Github

Git - It is a tool for controlling files or folders through commands

Workspace

Github - repository of files stored at cloud

Git commands

  1. git init - To initialize .git folder
  2. git status - To view the status of the file
  3. git add filename - To move the file to the staging area
  4. git commit -m “First Commit” - To commit the file
  5. git push origin master - To move the file to the remote repository
